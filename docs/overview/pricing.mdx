## Self-hosting

The OSS engine is **free forever** under Apache 2.0. Run it on your own infrastructure:

```bash
git clone https://github.com/bfortunato/sifter
cd sifter/code
cp .env.example .env  # add SIFTER_LLM_API_KEY
docker compose up -d
```

You bring your own LLM API key (OpenAI, Anthropic, Google, or Ollama). No per-document fees, no usage limits.

See [Self-hosting](/self-hosting/docker-compose) for the full setup guide.
